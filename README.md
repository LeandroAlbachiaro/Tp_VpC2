## Descripción del problema
En el entorno industrial la seguridad y eficiencia son fundamentales, por eso, un modelo utilizado para detectar personas y autoelevadores mediante modelos avanzados de Machine Learning puede ser de gran utilidad para  prevenir accidentes, mejorar la planificación logística y garantizar un entorno industrial más seguro.  

![enter image description here](https://user-images.githubusercontent.com/48186387/235376590-d4282565-1d1b-48fb-9345-64a4d969b3ff.png)

En este trabajo se explora la detección de personas y elevadores con las arquitecturas YOLO y Retinanet.

Dataset utilizado: [Roboflow - Forklift-dsitv](https://universe.roboflow.com/mohamed-traore-2ekkp/forklift-dsitv/dataset/2)
